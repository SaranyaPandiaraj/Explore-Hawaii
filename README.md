2) Back End Database : SQLite
3) Retrieving Data From Back End : Python (SQLAlchemy and Flask)
4) Data Visualization Libraries : D3 Js, Plotly Js ,Zing Chart, Leaflet Js
5) Deploying Application : Heroku
